3.5 vim nginx.conf进行编辑，主要添加如下两项

解决第一个问题：
location / {
    root /root/whc/www/shangpinhui/dist;
    index index.html;
    try_files $uri $uri/ /index.html;
}

location /api{
    proxy_pass:http://gmall-h5-api.atguigu.cn
}
